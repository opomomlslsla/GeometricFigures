# GeometricFigures

Написана библиотека классов позволяющая считать площадь и периметр фигур, чтобы добавить новую фигуру нужно добавить фигуру(класс) в папку Figures а также указать данную фигуру в перечислении FigureType и добавить условие в switch case в фабрике.
Также написан примитивный логгер, который можно заменить любым другим логгером при желании.
